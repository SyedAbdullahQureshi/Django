
# Django Todo API

Simple Todo API in Django without HTML templates.

## Setup
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## API Endpoints
- GET /api/todos/
- POST /api/todos/
- GET /api/todos/<id>/
- PUT /api/todos/<id>/
- DELETE /api/todos/<id>/
